Engineering PDF to Excel

Upload a PDF and get a cleaned Excel workbook.

This version uses a plain HTML interface instead of Streamlit. It extracts tables with pdfplumber, Camelot, and OCR fallback, then detects formula patterns and generates a downloadable XLSX file.

PDF extraction Native tables are read first, then scanned pages fall back to OCR when needed.
Formula detection Common engineering row patterns are converted into workbook formulas automatically.
Excel export Results are formatted and returned as a downloadable .xlsx workbook.
